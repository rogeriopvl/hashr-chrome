What is online and offline mode?

Click the online/offline button to switch between modes.

Offline mode allows you to create hashes offline (no request is made to a server).

In offline mode only a subset of hashing algorithms are available (this might change in the future).

Online mode creates hashes in a server (via api). It supports a wider range of hashing algorithms. Your privacy is respected, no info about your request is stored, and communication is done via https.
